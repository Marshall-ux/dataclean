¿Cómo uso la herramienta?

  1. 1. Exportá tu Excel o CSV desde el CRM de la concesionaria, facturación o Pilot.
  2. 2. Configurá las opciones (o dejá todo activado para usar los valores recomendados).
  3. 3. Arrastrá el archivo al recuadro principal de esta misma pantalla.
  4. 4. Descargá el reporte y abrí la pestaña "Indicadores" para leer los datos automáticos de cada columna.

Preguntas Frecuentes

¿Qué tamaño o cantidad de filas soporta?

El sistema puede procesar de forma robusta hasta 100.000 filas sin generar cuelgues. Para archivos muy grandes se recomienda dividir en hojas de hasta 50.000 filas.

¿Tengo que configurar qué columnas limpiar manualmente?

No. El sistema detecta automáticamente el tipo de cada columna (email, teléfono, fecha, moneda, categoría) y aplica la normalización correspondiente sin intervención manual.

¿Qué hace exactamente la pestaña Indicadores?

Genera un resumen automático por columna: registros por mes para fechas, distribución de estados, las 5 categorías más frecuentes, sumas y promedios de montos, y dominios de correo más usados.

¿Los archivos quedan guardados en el servidor?

No. Tanto el archivo original como el procesado se eliminan automáticamente del servidor al terminar la descarga. Tus datos no se almacenan.
